项目简介 
本系统是一个面向长尾噪声标签数据的可视化分析工具。系统参考TABASCO方法的核心思想，通过计算加权JS散度（WJSD）和自适应质心距离（ACD）两个度量，对样本进行两阶段选择，将样本分为高可信、低可信和疑似噪声三类。系统支持CIFAR-10N真实噪声数据集和CIFAR-10-LT模拟数据集，提供长尾分布可视化、噪声检测分析与特征空间PCA可视化、混淆矩阵、交互式标签修正等功能。

系统功能 
功能模块	说明
数据集加载	支持CIFAR-10N（真实噪声）、CIFAR-100N、Animal-10N、CIFAR-10-LT（模拟）
长尾采样	支持对真实数据集按不平衡率构造长尾分布（指数衰减公式）
TABASCO检测	两阶段样本选择，按一定规则划分并输出
高可信/低可信/疑似噪声三类
可视化分析	类别分布图、混淆矩阵、置信度分布、特征空间PCA可视化
交互修正	支持逐样本标签修改、批量修正、结果导出CSV

使用流程 
1)准备好数据集以及模型文件并置于相应位置，运行程序 bash streamlit run main.py 浏览器会自动打开 http://localhost:8501。
2)选择数据集，设置样本数量、噪声类型等参数和选项，点击加载数据集，待数据集加载成功后选择模型，点击TABASCO检测。

3)查看标签页。数据分布、噪声检测置信度部分、特征空间可视化部分、模型性能页面、交互模块即标签修改页面。上述所有图表均基于 Plotly 框架绘制。该框架无需额外编写代码即可为每个图表提供了丰富的交互功能，如，图例筛选，单击图例中的某一项（如“高可信”或特定类别），可以单独显示或隐藏对应的数据系列，便于聚焦分析特定类型的样本，双击图例可恢复全部显示；工具栏操作，鼠标悬停在图表右上角会显示工具栏，支持缩放（框选/自动缩放）、平移、重置视图以及下载为 PNG 图片等操作，方便用户保存分析结果；悬停提示，鼠标移动到数据点上会弹出提示框，显示该点的具体数值（如 PC1/PC2 坐标、样本类型、类别标签等）。这些交互特性可提升了系统的可操作性和用户体验，用户可在“数据分布”、“噪声检测”、“模型性能”等页面中使用上述功能以更深入地研究数据特征和模型行为。对于标签修改页面，额外筛选与排序、标签修正、批量操作、导出csv等功能。

导出CSV的列名及说明如下
样本ID	样本唯一标识
真实标签	真实类别（clean_label）
原始噪声标签	加载时的噪声标签
修改后标签	用户修正后的标签
预测标签	模型预测的类别
预测是否正确	预测结果是否为真实标签
置信度	预测置信度
样本类型	高可信/低可信/疑似噪声
原始是否为噪声	原始标签是否错误
修改是否正确	修正后是否正确
